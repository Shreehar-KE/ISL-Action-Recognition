## Install dependencies
### Local:
```
Linux:
python3 -m venv venv
source venv/bin/activate
pip install tensorflow opencv-python mediapipe

Mac:
pip install virtualenv
virtualenv venv
source venv/bin/activate
pip install tensorflow opencv-python mediapipe
```
